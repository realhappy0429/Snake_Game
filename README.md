# 贪吃蛇游戏 - C++ EasyX 图形版

一个基于 C++ 和 EasyX 图形库开发的经典贪吃蛇小游戏，界面美观、操作流畅，支持重新开始、分数统计、碰撞检测等功能。

## 🎮 游戏功能
- 方向键控制蛇上下左右移动
- 吃到食物自动变长
- 实时分数显示
- 撞墙 / 撞到自身游戏结束
- 按 **空格键** 重新开始游戏
- 精美游戏界面优化

## 🧰 开发环境
- 编程语言：C++
- 图形库：EasyX
- IDE：Visual Studio 2019 / 2022
- 系统：Windows

## 🚀 运行方法
1. 安装 EasyX 图形库
2. 使用 Visual Studio 打开项目
3. 直接编译运行
4. 方向键控制，空格重新开始

## 📷 界面预览
- 游戏主界面：蛇移动、食物、分数
- 游戏结束界面：显示最终分数，支持重新开始
